**FrankWolfe.jl** is an open-source Julia package for first-order constrained optimization using Frank-Wolfe (Conditional Gradient) methods.

Key features:
- High performance with Julia's multiple dispatch
- Flexible API supporting custom oracles
- Smooth integration with MathOptInterface.jl
- Implements multiple FW variants: vanilla, away-step, blended, lazified, and more

Published in the *INFORMS Journal on Computing* (2022).

[Paper](https://arxiv.org/abs/2104.06675) · [GitHub](https://github.com/ZIB-IOL/FrankWolfe.jl) · [Documentation](https://zib-iol.github.io/FrankWolfe.jl/stable/)
